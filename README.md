# Practice with data manipulation:
A simple web page that manipulates data from an external API and shows it.

![page demonstration](./assets/demo.gif)

## Objectives:
- 1- Fetch the data form the API  https://api.origamid.dev/json/transacoes.json

- 2- Show in a table the data of each transaction

- 3- Calculate and show the statics:
  - 3.1- Total sum of the values
  - 3.2- Transactions for payment method
  - 3.3- Transactions per status
  - 3.4- Total selling amount per week day
  - 3.5- The day of the week with the most sells
